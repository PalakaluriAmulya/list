1. What is HTML ?
      2. Def : HTML (HyperText Markup Language) is the standard language used to create and structure content on the web, like text, images, links, and more.
  2. What is the difference between HTML & XHTML ?
      3. HTML
    • HTML is the standard language used to create and design webpages.
    • It is flexible and allows small mistakes without breaking the page
    • Tags in HTML do not always need to be closed, and case doesn’t matter.
    • It is widely used today, especially in HTML5, for building modern websites
      • XHTML
    • XHTML is a stricter version of HTML based on XML rules
    • It requires all tags to be properly closed and written in lowercase.
    • Attribute values must be enclosed in quotes, and no errors are allowed.
    • XHTML is used when XML compatibility and strict structure are required.
  3. What is tag ?
      4. Def : A tag in HTML is a piece of code that defines the structure or content of a webpage.
  4. What is element ?
      5. Def : An HTML element is everything from the start tag to the end tag, including the content in between.
      < start tag >hello < ending tag >
  5. What is the difference between HTML4 & HTML5 ?
      6. =>HTML4 (Old Version) :
    • Released in 1997.
    • Used for structure, but relied heavily on CSS or JavaScript for features like animation, video, etc.
    • Didn't support multimedia (like audio/video) directly.
    • Was not mobile-friendly.
    • Required extra plugins (like Flash) for interactive content
      • =>HTML5 (Current Version):
    • Released in 2014.
    • Supports multimedia elements like < audio >, < video >, < canvas > directly.
    • Mobile-friendly and better for responsive design.
    • Includes semantic tags like < header >, < footer >, < article >, and < section > to improve SEO and code readability.
    • Built-in form validation and modern APIs (like geolocation, localStorage).

  7. What is difference between XML and HTML ?
      HTML (HyperText Markup Language)
    • Used to design web pages and display content
    • Has predefined tags like < h1 >, < p >, < div >, etc.
    • Focuses on how data looks (formatting, structure).
    • Not strict about syntax.
    • Tags are mainly for presentation
      XML (eXtensible Markup Language)
    • Used to store and transport data.
    • You must define your own tags.
    • ocuses on what data means (structure and meaning).
    • Very strict about syntax (must be well-formed).
    • No predefined tags — it's not for display, just for data
  8. What are the attributes ?
    • In HTML, attributes are special words used inside the opening tag of an element to control the element's behavior or provide additional information about it.
    • Key Points About HTML Attributes:
        • Key Points About HTML Attributes:
      1. They are written inside the opening tag.
      2. Usually come in name/value pairs, like: name="value".
      3. Most attributes are optional, but some are required depending on the element.
        4. Attribute Discription
        id Assigns a unique identifier to the element
        class Assigns one or more class names (used with CSS or JS)
        style Inline CSS styling
        href Specifies the URL for a link (< a >)
        src Specifies the source file for media like images (< img >)
        alt Alternative text for images
        value Sets the default value for form elements
